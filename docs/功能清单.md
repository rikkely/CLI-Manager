> 更新时间：2026-06-08  
> 参考来源：`README.md`、`CHANGELOG.md`、当前前端与 Tauri/Rust 后端源码只读梳理。

## 1. 产品定位

CLI-Manager 是一款面向 Windows 开发者的多项目 CLI 工作台，核心目标是把多项目开发中的 **项目、终端、命令、AI CLI 会话、历史回看、统计与同步** 集中到一个桌面应用中管理。

当前项目已经覆盖：

- 多项目与分组管理
- 内置 PTY 终端
- Tab 与 JetBrains 风格分屏
- Claude Code / Codex CLI Hook 状态桥接
- 命令模板与命令历史
- Claude / Codex 历史会话回看
- Diff 查看
- 分析看板
- WebDAV 云同步
- 主题、快捷键、终端背景等个性化设置

## 3. 内置终端工作区

### 3.1 PTY 终端能力

- 基于 Rust 后端 PTY 创建真实终端会话
- 前端使用 xterm.js 渲染终端
- 支持终端输入写入 PTY
- 支持终端尺寸 resize 同步
- 支持关闭 PTY 会话
- 支持查询终端进程状态
- 后端输出按安全边界切分，避免截断 UTF-8 字符或 ANSI 转义序列
- PTY resize 有最小尺寸保护，避免不可见容器导致异常尺寸

### 3.2 Tab 管理

- 新建终端 Tab
- 关闭当前 Tab
- 重命名 Tab
- Tab 拖拽排序
- Tab 溢出滚动
- Tab 溢出列表控制
- 关闭其它 Tab
- 关闭左侧 Tab
- 关闭右侧 Tab
- 切换 Tab 时保留终端缓冲区
- 隐藏 Tab 有界输出缓冲，降低后台占用

### 3.3 终端状态展示

- Tab 状态点
- 项目树聚合状态点
- 状态类型：
  - 运行中
  - 待审批
  - 已完成
  - 异常退出
- 状态优先级：待审批 > 异常退出 > 运行中 > 已完成
- PowerShell / pwsh 命令运行监控
- Shell runtime monitoring 可在设置页关闭

### 3.4 终端输出与输入优化

- 高频 PTY 输出合并后限频写入 xterm
- 非激活终端降频刷新
- WebGL 渲染失败或上下文丢失时回落 Canvas 渲染
- 中文输入法组合输入期间暂停自动 fit，降低界面抖动
- 终端粘贴支持
- 终端复制支持
- Codex 高频重绘场景下合并光标显示序列，减少闪烁
- 终端输出搜索浮层
- `Ctrl+F` 搜索终端输出
- 搜索结果高亮
- 上一个 / 下一个搜索结果导航
- 关闭搜索后恢复终端焦点

## 6. Claude Code / Codex CLI 集成

### 6.1 Hook Bridge

- 应用启动本地回环 Hook 通知服务
- 仅监听 `127.0.0.1`
- 使用 bearer token 校验 Hook 请求
- 为每个 PTY 会话注入：
  - `CLI_MANAGER_TAB_ID`
  - `CLI_MANAGER_NOTIFY_PORT`
  - `CLI_MANAGER_NOTIFY_TOKEN`

### 6.2 Claude Code Hook

支持事件：

- `UserPromptSubmit`
- `Notification`
- `Stop`
- `StopFailure`

能力：

- Claude 提交用户 Prompt 时标记对应 Tab 为运行中
- Claude 需要处理时标记对应 Tab 为待审批
- Claude 完成时标记对应 Tab 为已完成
- Claude 失败时标记对应 Tab 为异常退出
- 支持悬浮通知卡片
- 切换到目标 Tab 后自动清理通知

### 6.3 Codex CLI Hook

支持事件：

- `UserPromptSubmit`
- `PermissionRequest`
- `Stop`

能力：

- Codex 提交用户 Prompt 时标记对应 Tab 为运行中
- Codex 权限请求时标记对应 Tab 为待审批
- Codex 停止时标记对应 Tab 为已完成
- 与 Claude 共用终端 Tab 通知逻辑
- 按来源区分 Claude / Codex 通知

### 6.4 Hook 设置

- Claude / Codex Hook 设置拆分展示
- 支持选择 Claude 配置目录
- 支持选择 Codex 配置目录
- 支持安装 Claude Hook
- 支持卸载 Claude Hook
- 支持安装 Codex Hook
- 支持卸载 Codex Hook
- 展示 Hook 安装状态
- 展示脚本路径
- 展示配置文件路径
- Claude Hook 安装时合并写入 `settings.json`，不删除用户自定义 Hook
- Codex Hook 安装时写入 `hooks.json`
- Codex Hook 安装时确保 `config.toml` 中 `[features].hooks = true`
- Hook 弹框全局开关
- Hook 弹框自动关闭时间配置

- 背景图片按 SHA-256 内容寻址命名

## 20. 安全与边界处理

### 20.1 历史会话安全

- 删除历史会话前校验来源
- 删除历史会话前校验项目边界
- 删除历史会话前 canonicalize 路径
- 只允许删除受支持来源下的 `.jsonl` 文件
- 切换 Claude / Codex 配置目录后隔离历史索引与缓存

### 20.2 Hook Bridge 安全

- Hook 服务仅监听本地回环地址
- Hook 请求必须携带 bearer token
- Hook body 有大小限制
- Hook header 有大小限制
- 只接受指定事件类型

### 20.3 背景图片安全

- 图片来源必须是绝对路径
- 只允许 `jpg` / `jpeg` / `png` / `gif`
- 禁止 `webp`
- 保存后的相对路径禁止：
  - 空路径
  - `..`
  - 反斜杠
  - 绝对路径
  - 逃逸 `backgrounds/` 目录
- asset protocol scope 锁定到背景图目录

### 20.4 WebDAV 安全与稳定性

- 使用 Basic Auth
- WebDAV 响应大小限制
- 请求超时限制
- 上传前确保远端目录存在
- 旧路径兼容但新数据使用设备快照路径

### 20.5 PTY 稳定性

- PTY 输出避免切断 UTF-8 字符
- PTY 输出避免切断 ANSI / OSC / CSI 转义序列
- 未完成残尾延迟到下一轮拼接
- 异常源端有缓冲兜底，避免无限增长
- resize 有最小列数 / 行数保护
